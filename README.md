

# 📊 Sistema Automatizado de Análisis de Ventas

Este proyecto consiste en un script en Python que automatiza el procesamiento de datos de ventas a partir de un archivo CSV, generando métricas clave, reportes y visualizaciones de manera automática.

---

## 🚀 Tecnologías utilizadas

- Python
- Pandas
- Matplotlib

---

## 📁 Estructura del proyecto

```

proyecto_analisis/
│
├── data/
│   └── ventas.csv
│
├── output/
│   ├── reporte.txt
│   └── grafico.png
│
├── script.py
└── requirements.txt

````

---

## 📦 Instalación

Clona el repositorio y ejecuta:

```bash
pip install -r requirements.txt
````

---

## ▶️ Ejecución

```bash
python script.py
```

---

## 📊 Funcionalidades

* Lectura de datos desde archivo CSV
* Limpieza de datos (eliminación de valores nulos)
* Cálculo de métricas clave:

  * Total de ventas
  * Promedio de ventas
  * Top vendedores
* Generación automática de reporte en archivo `.txt`
* Creación de gráfico de ventas por categoría

---

## 📈 Resultados

El script genera automáticamente:

* 📄 `output/reporte.txt` → resumen de métricas
* 📊 `output/grafico.png` → visualización de datos

Demostrar habilidades en:
