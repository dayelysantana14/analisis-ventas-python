# 📊 Sistema Automatizado de Análisis de Ventas

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)
[![Pandas](https://img.shields.io/badge/Library-Pandas-150458.svg)](https://pandas.pydata.org/)
[![Seaborn](https://img.shields.io/badge/Library-Seaborn-3776AB.svg)](https://seaborn.pydata.org/)

Este proyecto consiste en un **Pipeline de Análisis de Datos** desarrollado en Python que automatiza el procesamiento de información de ventas. El sistema transforma datos brutos en reportes ejecutivos y visualizaciones estratégicas, eliminando la necesidad de procesamiento manual y errores humanos.

---

## 🚀 Tecnologías Utilizadas

- **Python**: Lenguaje principal para la lógica de automatización.
- **Pandas**: Limpieza, manipulación y análisis estadístico de los datos.
- **Seaborn & Matplotlib**: Generación de visualizaciones profesionales y storytelling de datos.
- **OS**: Gestión programática de directorios y rutas de archivos para asegurar la portabilidad del proyecto.

---

## 📁 Estructura del Proyecto

```text
smart-sales-analytics/
│
├── data/               # Archivos fuente (CSV)
│   └── ventas.csv
├── output/             # Resultados generados automáticamente
│   ├── reporte_ventas.txt
│   └── grafico_pro.png
├── script.py             # Pipeline de análisis principal
└── requirements.txt    # Dependencias del proyecto
```
----
## 📊 Funcionalidades Clave

  - **Data Cleaning**: Tratamiento de valores nulos y estandarización de formatos de fecha/moneda.

  - **Análisis Descriptivo**: Cálculo de volumen total, promedio de transacciones y ranking de performance.

  - **Reporte Automático**: Generación de un archivo .txt con métricas listas para auditoría.

  - **Visualización**: Gráfico de barras con estilo moderno para identificar tendencias por categoría.
----

## ⚙️ Instalación y Uso

   **Clonar repositorio**:
    
  ```Bash

    git clone [https://github.com/dayelysantana14/analisis-ventas-python.git](https://github.com/dayelysantana14/analisis-ventas-python.git)

  ```

   **Instalar dependencias**:
     
   ```Bash

    pip install -r requirements.txt
   ```
   **Ejecutar Pipeline**:
 ```Bash

    python script.py
 ```
